## MMS Calc

This is an extension that would calculate your exam grades for you.

![screenshot here](readme_assets/image.png)

Works both on [firefox](https://addons.mozilla.org/en-US/firefox/addon/mms-calc/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search) and [google-chrome](https://chromewebstore.google.com/detail/mms-calc/dopahpobcogpelhlepodioihbegakdmb)

Enjoy :)
